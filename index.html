S3 File Uploader

This web application lets you upload files directly to an Amazon S3 bucket using presigned URLs—no backend server required.

  • Upload to S3: Paste your presigned PUT URL, select or drag a file, and click Upload to send the file to S3.
  • Generate Sharable Link: If you already have a presigned URL and need to share it, paste it here and click Generate Sharable Link. You’ll get a link that opens this app with the URL prefilled for easy uploading.

Make sure your S3 bucket’s CORS settings and IAM permissions allow these operations.
